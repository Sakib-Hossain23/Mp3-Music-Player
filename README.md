## Mp3-Music-player

Mp3-Music-Player made with a simple HTML, CSS & JavaScript MP3 Player. This music player has several features like play/pause songs, and play next/prev songs. You can select songs from the file manager.

# Technologies Used:
HTML
CSS
JavaScript
